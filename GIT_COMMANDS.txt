# Quick Git Commands for Deployment

# Step 1: Add all your files
git add .

# Step 2: Commit your code
git commit -m "Initial commit - OSINT Web Application"

# Step 3: Add your GitHub repository (REPLACE with your actual GitHub URL)
# Example: git remote add origin https://github.com/YOUR_USERNAME/osint-web.git
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git

# Step 4: Push to GitHub
git branch -M main
git push -u origin main

# After this, go to Netlify and connect your GitHub repository!
